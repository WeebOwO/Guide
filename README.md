# 前言

本文档主要是介绍一些必要的技术基础以及我个人在学习过程中的经验分享，希望能够帮助新来到实验室的同学快速上手，减轻一些不必要的焦虑。

### 通过该文档你会掌握什么？

本文档并不是事无巨细的技术文档，例如Python或者Pytorch中某个API的具体用法并不会在这里被提及。不过，我会相应详细的介绍你需要哪些知识来顺利的进行后续研究，并且给出我在学习这些知识时的一些看法以及对应的学习方式。
